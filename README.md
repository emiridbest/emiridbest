## Biomolecular ML Research

Medically trained (MBBS) data scientist with published research in clinical ML and active MSc dissertation work in deep learning for protein science.

- **PPI Prediction for Alzheimer's Disease** — Multimodal pipeline: transformer protein embeddings (ESM/ProtBERT) + MSA evolutionary features + cavity-level structural descriptors. Positive-Unlabelled classifier: AUC-ROC **0.9965**, AUPR **0.9989**. *(Preprint in preparation)*
- **Generative Protein Design** — Conditional Wasserstein GAN for biologically plausible protein sequence generation conditioned on AD-associated proteins
- **Published** — [*Predictive Modelling for Thyroid Cancer Recurrence*](https://github.com/emiridbest/thyroid_CA_prediction), Springer LNNS (ICDPN-2025)
- **Tools** — PyTorch · Hugging Face · ESM/ProtBERT · Biopython · gemmi · RDKit · AlphaFold2 · scikit-learn · PySpark
